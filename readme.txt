모든 좌표를 배정해서 주는 노가다를 실시하였고...총 vao3개로 vao0은 좌표축, vao1은 육면체, vao2는 사면체입니다.
강의에 나와있는대로 실행했습니다.

키설정
1~6 : 육면체 한면씩 그리기, 7~0 : 사면체 한면씩 그리기
a~c : 육면체 서로 반대방향의 두면 그리기
e~g : 사면체 밑면 + 각 옆면 그리기

14.

13에서 썼던 좌표에 추가하여 사용했고, 처음 실행 시 좌표축만 나오게 됩니다.
c나 p키를 눌러야 나오게 됩니다. c,p키를 누를때마다 도형들은 각도와 이동하는 좌표 초기화 시켜줍니다.
s키도 c,p키와 비슷하지만 해당 도형 그대로인 상태로 각도와 이동좌표만 초기화 합니다.
h키를 누를 때마다 은면제거와 제거 해제가 번갈아 가며 실행됩니다. 초기엔 은면제거가 해제 되어있습니다.
자전하는 키 중 같은 키를 두번 누르게 되면 정지합니다.
음이나 양의 방향으로 회전 중, 반대 방향 회전하는 키를 누르게 되면 자연스럽게 반대 방향으로 회전합니다.
special키 함수를 통해 이동하는 것을 구현하였습니다.

키설정
c : 육면체 p : 사각뿔 h : 은면제거/해제
x : 양의 방향으로 회전/해제
X : 음의 방향으로 회전/해제
y : 양의 방향으로 회전/해제
Y : 음의 방향으로 회전/해제
w : 와이어모드
W : 솔리드모드
s : 초기위치로 리셋 (도형 유지)
방향키 up/left/down/right 방향으로 이동 (x,y축으로 이동)

15.

13에서 썼던 좌표에 추가하여 사용했고, 실행 시 좌표 축과 사각뿔, 육면체가 등장합니다.
공전계산 이동계산 자전계산 순으로 실시 하였고, 회전 시 14와 같이 두번 누르게 되면 정지,
회전 중 회전과 반대방향으로 회전하면 반대로 돌아가게 하였습니다.
솔리드 모드로 하면 도형보기가 불편할 수 있어서 와이어모드로 변환하였습니다.
c키를 눌러 두 도형을 다른 두 도형으로 바꿉니다. 이때 사각뿔과 육면체는 각각 구와 원기둥으로 그려집니다.
도형을 바꿀때, 기존에 회전하는 값들은 14와 다르게하여 초기화 하지 않습니다.
도형을 바꾸고 나서 회전키를 누르면 해당 회전에서 회전 키 방향으로 회전합니다.
s키를 눌렀을때 모두 초기화합니다.

키설정

x : 양의 방향으로 회전/해제 (구와 육면체)
X : 음의 방향으로 회전/해제 (구와 육면체)
y : 양의 방향으로 회전/해제 (구와 육면체)
Y : 음의 방향으로 회전/해제 (구와 육면체)
a : 양의 방향으로 회전/해제 (원기둥과 사각뿔)
A : 음의 방향으로 회전/해제 (원기둥과 사각뿔)
b : 양의 방향으로 회전/해제 (원기둥과 사각뿔)
B : 음의 방향으로 회전/해제 (원기둥과 사각뿔)
r : 양의 방향으로 회전/해제
R : 음의 방향으로 회전/해제
c : 두 도형 다른 두 도형으로 교체
s : 회전값 초기화 (도형이 초기에 있던 자리로 돌아감)
